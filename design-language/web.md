# Web

The 7CS website should feel like a digital library for an institution.

## Principles

- Quiet before loud.
- Editorial before promotional.
- Durable before trendy.
- Precise before decorative.

## Reading Experience

The Bible should be presented as a canonical reading path with clear chapter order, previous and next navigation, and generous spacing.

## Visual Direction

Use restrained contrast, institutional typography and measured accents. Brushed gold should be used sparingly as emphasis, never as decoration.

## Website Rule

The website must preserve the source Markdown as the institutional record. Presentation may evolve, but the source of truth remains the Constitution repository.
